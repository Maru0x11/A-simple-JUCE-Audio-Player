# A simple Audio Player
This repository contains the second assignment for the CS213 course. 
A simple audio player capable of playing Audio files in different formats, looping audio tracks, changing speed, loading playlists of audio files into the program, A/B segment looping.
The project is written in c++ and the JUCE C++ Framework utilities.
